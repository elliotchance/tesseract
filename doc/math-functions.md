% Mathematical Functions


`ABS()` -- Absolute Value
=========================

Return the absolute value (positive value) of a number.

    abs(<number>)


`CEIL()` -- Round Up
====================

Round up to the next whole number.

    ceil(<number>)


`FLOOR()` -- Round Down
=======================

Remove the fractional part of a number. This is often refered to as truncating a
number.

    floor(<number>)
