String Functions
================


``BIT_LENGTH()`` -- Bit Length
------------------------------

Return the bit length of a string.

.. code-block:: sql

   bit_length(<string>)


``CHAR_LENGTH()`` -- Character Length
-------------------------------------

Return the character length of a string.

.. code-block:: sql

   char_length(<string>)


``OCTET_LENGTH()`` -- Octet Length
----------------------------------

Return the octet length of a string.

.. code-block:: sql

   octet_length(<string>)
